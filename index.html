Welcome in the GB4 :gb: The Hideout

This is the default Hideout for Gblade4. Here, we test all the features available by default* (see FEATURES in index.js).

Features requiring a web server

Some parts of the application will not work if opened directly from the local file.
To test all features, launch a web server, such as Live Server or a Node.js server, so that the application is served via HTTP.

List of Features

3D card Mouse Demo

Here are almost all the possibilities for 3D cards in relation to the position of the mouse.

Use the mouse-3d-transform class to then configure each behavior.
- data-sensibility adjusts the sensitivity of the tilt in relation to the distance of the mouse.
- data-hover determines whether it should tilt only when hovered over or not.
- data-lock allows you to lock a rotation on one or more axes.
- data-inverse reverses the tilt in relation to the position of the mouse.

All Default GB CSS import variables

Here are all the variables present in the CSS. If some are modified when they have already been declared, the display of the variables depends on which loads first.

Use getCSSVariables() to retrieve all CSS variables.

Tabs system

Add buttons with data-tab-collection and data-tab-btn, and panels with data-tab-collection and data-tab. When you click a button, all panels in that collection are hidden, and the one matching the button’s ID is shown using flex.
